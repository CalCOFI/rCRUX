save_output_as_csvR Documentation

Function to save data in .csv files

Description

Calls write.table with file_name as the first argument and <file_out><Metabarcode><description>.csv as the second

Usage

save_output_as_csv(file_name, description, file_out, Metabarcode)

Arguments

file_name

the object to write

description

a string to include in the .csv name directly before the extension

file_out

a parent directory; it should end in a /

Metabarcode

a label that appears before the file name

Value

The value of write.table
