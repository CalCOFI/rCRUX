run_blastnR Documentation

Runs blastn with the input fasta as a query

Description

Takes the input string, writes it to a temporary file, and calls blastn with that file as the query. Warning: if a file at the path specified by temp already exists, it will be overwritten then deleted.

Usage

run_blastn(
  fasta,
  db_dir,
  temp = NULL,
  ncbi_bin = NULL,
  evalue = 1e-06,
  align = 50000,
  coverage = 50,
  perID = 70
)

Arguments

fasta

a fasta-formatted string

temp:

a file path to write a temporary fasta to

ncbi_bin:

if not null use it as the parent directory for blastn

Value

a tibble representing the blastn results
