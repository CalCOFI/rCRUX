Description

Wrapper function for sample If the sample size is larger than the population, returns the input vector Otherwise returns the result of sampling

Usage

smart_sample(x, size, ...)

Arguments

x

a vector to sample from

size

the size of the sample

...

additional arguments to pass to sample

Value

a vector sampled from x
