filter_primer_hitsR Documentation

Remove empty accession numbers and other problematic rows

Description

Remove empty accession numbers and other problematic rows

Usage

filter_primer_hits(
  hits_table,
  forward_primer,
  reverse_primer,
  mismatch = 3,
  minimum_length = 5,
  maximum_length = 500
)

Arguments

hits_table

a data.frame generated by parse_primer_hits or by iterative_primer_search

mismatch

the maximum allowable value (inclusive) in the mismatch_forward and mismatch_reverse columns

Value

a data.table with problematic rows removed
